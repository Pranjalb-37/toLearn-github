# Java Sorting Practice

This repository contains Java programs for learning and practicing sorting algorithms.

## Files

* `bubbleSort.java` — Bubble Sort implementation
* `selectionSort.java` — Selection Sort implementation
* `insertionSort.java` — Insertion Sort implementation
* `cyclicSort.java` — Cyclic Sort implementation
* `Solution.java` — Additional practice problems / solutions

## Purpose

This repository is created to practice Java programming and improve problem-solving skills and understand sorting techniques.

## Language

* Java

## How to Run

1. Clone the repository
2. Open the project in VS Code or IntelliJ
3. Compile and run a Java file

Example:

```bash
javac insertionSort.java
java insertionSort
```
